基于github+Hexo创建个人博客

发表于 2022-03-03

由于本身自己本科电子信息，对于网络方面，计算机方面的知识比较匮乏。故结合自身在写个人博客方面的经验写一个对0基础人群的博客构建文章。

博客语法

推荐使用markdown进行博客的书写，windows端推荐使用typora，相关下载以及使用相关问题请自行搜索。markdown语法入门可在bilibili或菜鸟教程进行搜索。

博客书写以及第一篇文档上传
